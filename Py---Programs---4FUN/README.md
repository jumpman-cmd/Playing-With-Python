# Symmetric Difference Calculation

This repository contains a Python program to calculate the symmetric difference between two sets of integers. The symmetric difference includes elements that are in either of the sets, but not in both.

## Problem Statement

Given two sets of integers, `A` and `B`, the task is to print their symmetric difference in ascending order. The symmetric difference indicates those values that exist in either `A` or `B`, but do not exist in both.

### Example

**Input:**
4
2 4 5 9
4
2 4 11 12

**Output:**
5
9
11
12

The input is read from standard input and processed to extract the two sets of integers. The symmetric difference is computed using Python's built-in set operations, and the result is printed in ascending order.
