# Consecutive Integer Printer

## Task
Given an integer `n`, print the consecutive values from 1 to `n` as a single string without spaces.

## Example

### Input
3

### Output
123

## Input Format
- A single line containing an integer `n`.

## Constraints
- `1 <= n <= 150`

## Output Format
- Print the list of integers from 1 through `n` as a string, without spaces.

## Solution

This Python program reads an integer `n` from standard input and prints the consecutive integers from 1 to `n` as a single string without spaces.
