# Consecutive Integers as a String

## Problem Explanation

The task is to print the consecutive integers from 1 to \( n \) as a single string without any spaces in between. The constraint is to achieve this without using any string methods.

### Input Format

- The first line contains an integer \( n \).

### Constraints

- \( 1 \leq n \leq 150 \)

### Output Format

- Print the list of integers from 1 through \( n \) as a string, without spaces.

### Example

#### Sample Input 0
3

Sample Output 0
123
